# didit

<https://didit.iwamot.com/> is a site for those who have passed [Japan's Information Technology Engineers Examination](https://www.jitec.ipa.go.jp/).

## How to Join

1. Fork this repository
2. Create data/{YOUR_GITHUB_ID}.json referring to [data/iwamot.json](https://github.com/iwamot/didit/blob/master/data/iwamot.json)
3. Send a pull request

All pull requests are welcome, including features and design improvements.

## Build Setup

```bash
# install dependencies
$ yarn

# serve with hot reload at localhost:3000
$ yarn dev

# export static HTML
$ yarn build

# serve static files locally
$ yarn serve

# test
$ yarn test:e2e

# lint (check formatting and linting rules)
$ yarn lint

# fix (auto-fix formatting and linting issues)
$ yarn fix
```

## Linting & Formatting

This project uses [Biome](https://biomejs.dev/) for linting and formatting.

For detailed explanation on how things work, see [Next.js docs](https://nextjs.org/).
